# 半身分离游戏 Demo

一个基于 HTML5 Canvas 的 2D 像素风游戏原型。

## 游戏机制

- **下半身（红色）**：负责移动（A/D键）
- **上半身/头（黄色）**：负责瞄准和发射火球

## 操作说明

| 按键 | 功能 |
|------|------|
| **A/D** | 左右移动 |
| **短按空格** | 跳跃 |
| **长按空格** | 分离上半身并发射（抛物线）|
| **左键** | 发射火球（始终从头发射）|
| **分离后短按空格** | 融合（需靠近）|
